# James

James é um ERP pessoal, de usuário único e auto-hospedado. 

Os ícones do projeto são provenientes do [Heroicons](https://heroicons.com/).

## Módulos Planejados

- Finanças (inspirado no [Aurum2.0](https://github.com/ArthurWillers/Aurum2.0))
- Imposto de Renda
- Patrimônio
- Documentos
- CRM Pessoal
- Tarefas/Projetos
- Hábitos
- Saúde
- Despensa
- Receitas Culinárias
- Manutenção
- Viagens
- Conhecimento
- Dashboard
